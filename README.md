# Product Recommendation System

A machine learning system that predicts how likely a product is to be recommended, with full explainability using SHAP and LIME. Built on the Amazon Unlocked Mobile Phones dataset (400k reviews).

## What it does

- Predicts recommendation likelihood (0-100%) for a product
- Explains why using SHAP force plots and LIME bar charts
- Generates a plain English explanation using Groq LLM
- Identifies bias across price groups, brand size, and sentiment

## How to Use the App

1. Select a **brand** from the dropdown
2. Enter the **price** of the product
3. Select the **price tier** (Budget, Mid-range, Premium, Ultra-premium)
4. Type a **product review** in the text box — write it as if you are a customer
5. Click **Predict**
6. The app will show:
   - Recommendation likelihood as a percentage
   - A SHAP force plot explaining which features drove the score
   - A LIME chart showing feature contributions
   - A plain English explanation generated by AI

## Run Locally

### 1. Clone the repo
git clone https://github.com/FeaRleSSLegend/xai-project.git
cd xai-project

### 2. Install dependencies
Make sure you have uv installed, then run:
uv sync

### 3. Get a Groq API key
- Go to https://console.groq.com
- Sign up for a free account
- Navigate to API Keys and create a new key

### 4. Add your API key
Create a file at .streamlit/secrets.toml and add:
GROQ_API_KEY = "your_groq_api_key_here"

### 5. Run the app
uv run streamlit run app.py

The app will open at http://localhost:8501

## Stack

- XGBoost — classification model
- SHAP — global and local explainability
- LIME — local explainability
- VADER — sentiment analysis
- Groq (llama-3.3-70b-versatile) — plain English explanations
- Streamlit — deployed interface

Amazon Unlocked Mobile Phones via Kaggle
